Bases de données

31.01.2020

Installation

Il faut installer Mysql Workbench.

Développé par Oracle, MySQL Workbench est un logiciel gratuit qui permet de gérer et d'administrer des bases de données MySQL via une interface graphique.

Les requetes principales

SELECT * FROM nomTable WHERE attr1="salut"
INSERT INTO nomTable VALUES (attr1, attr2)
UPDATE nomTable set attr = monAttr WHERE attr = "value"
DELETE FROM nomTable WHERE attr = "value"

les fonctions de groupe

  • AVG
  • COUNT
  • MAX
  • MIN
  • SUM

Autres fonctionnalités

  • LIKE

les clauses

  • ORDER BY
  • HAVING

les jointures

  • Natural Join
    Permet d'associer 2 tables qui ont une colonnes identique (ex : num_dpt dans Employé et num_dpt dans Departement)
  • Inner Join (ou Join)
    Permet de dire sur quels champs les tables sont associées (ON table.attr = table2.attr).
  • Auto Jointure
    Il s'agit d'une association reflexive, la table est associée à elle même. La clé étrangère est sur la même table (ex : manager_id).
  • Jointure multiple
    Il s'agit d'une requete avec plusieurs jointures, il faut respecter l'ordre de liaison suivant le diagramme UML (ex job JOIN employe ON ... JOIN Departement ON ...)
  • Non-equi jointure
    C'est une jointure sur une inégalité avec un BETWEEN.
  • Jointure externe (outer join/left outer join
    Il s'agit du fait d'avoir des lignes qui n'ont pas forcement de correspondance avec les lignes de l'autre table.
Les alias
FROM table A inner join table2 B
